# COMPONENTS

This directory contains hardware components, sensors, and related documentation.

## Contents

### Display/SSD1306
SSD1306 OLED display module implementations and examples for various platforms:
- STM32F303 examples
- NXP FRDM-K22F examples
- Library implementations and drivers

### Liniar Sensor
Linear sensor datasheets and documentation (TSL2014).

### WS2812 / WS2812B
Addressable RGB LED strip datasheets and documentation.

### YF923
YF923 component documentation and images.
