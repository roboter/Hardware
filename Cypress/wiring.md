# Cypress PSoC Wiring and Programming

Cypress PSoC (Programmable System-on-Chip) microcontrollers combine MCU cores with programmable analog and digital blocks.

## Programming
- **Interface**: SWD (Serial Wire Debug).
- **Tools**: MiniProg3, MiniProg4, or on-board KitProg.
- **Software**: PSoC Creator or ModusToolbox.

## Specific Boards
- [CY8CKIT-059/](CY8CKIT-059/): PSoC 5LP Prototyping Kit.

## Connection Reference (SWD)
| Pin | Function |
| --- | --- |
| VTARG | Target Power |
| GND | Ground |
| SWDIO | Serial Wire Data |
| SWXCLK | Serial Wire Clock |
