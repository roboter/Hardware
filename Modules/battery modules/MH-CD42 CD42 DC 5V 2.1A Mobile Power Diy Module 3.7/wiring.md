# Wiring — MH‑CD42 (5 V boost module)

Typical pins (confirm on board):
- BAT / VIN : battery + (3.7–4.2 V)
- GND : battery - and system ground
- VOUT / 5V : boosted 5 V output to USB / load
- KEY / EN : optional enable / pushbutton input

Example:
Battery + ---- [BAT]
Battery - ---- [GND]
Load +  ------ [VOUT]
Load -  ------ [GND]

Notes:
- Verify rating before connecting heavy loads (label suggests ~2.1 A capability).
