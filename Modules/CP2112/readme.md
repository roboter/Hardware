# CP2112 USB‑to‑I2C Bridge Module

<img src="./s-l1600.jpg" width="320" alt="CP2112 module"/>

**Short description**
Silicon Labs CP2112 is a USB-to-I2C bridge that provides an easy USB interface to I2C devices. Useful for connecting I2C sensors and peripherals to a PC via USB.

**Key notes**
- USB host side: USB Type connector
- I2C side: SDA / SCL, VCC, GND
- Driver support: Silicon Labs CP2112 drivers and APIs
