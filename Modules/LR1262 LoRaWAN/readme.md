# LR1262 LoRaWAN Node Module

<img src="./LR1262_LoRaWAN_Node_Module_Datasheet.pdf" width="320" alt="LR1262 datasheet"/>

**Short description**
LR1262 family LoRaWAN nodes are wireless modules built around Semtech SX126x transceivers. Used for LoRaWAN network nodes, they typically expose SPI/UART interfaces for MCU integration and require a 3.3 V power supply and an antenna.

**Key notes**
- Radio IC: SX1261 / SX1262 (see datasheet)
- Frequency: region dependent (e.g., 433/868/915 MHz)
- Interface: SPI (radio control) and possibly UART for higher-level modules
