# RDA5807M FM Stereo Radio Module

<img src="./10719.jpg" width="320" alt="RDA5807M module"/>

**Short description**
RDA5807M is a small FM receiver module commonly used with microcontrollers (I2C control) to tune and receive FM radio, with stereo audio outputs.

**Key notes**
- Interface: I2C (SDA/SCL)
- Audio outputs: L/R analog outputs
- Power: 3.3 V typical (some modules support 5 V)
