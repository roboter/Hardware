  *  ESP32 dual core MCU
  *  4.7-inch integrated e-paper display - 540x960 pixels, 16 gray levels

  *  16MB flash
  *  8MB PSRAM memory
  *  USB-C port
  *  built-in LiPo/Li-Ion battery charger
  *  integrated 18650 Li-Ion battery holder
  *  3 general purpose buttons
