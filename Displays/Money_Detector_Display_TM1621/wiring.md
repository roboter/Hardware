# Wiring — TM1621 display

- Interface:
  - TM1621 typically uses a serial interface (DATA and CLK lines) with an optional strobe/enable.
- Power:
  - Commonly 5V supply — check module labeling.
- Notes:
  - See `Money_Detector_Display_TM1621.ino` for example connections and code.
- Safety:
  - Confirm VCC and logic levels before connecting to microcontrollers.
