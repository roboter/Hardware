# HD44780 Character LCD

Standard parallel character LCD modules based on the HD44780 controller (e.g., 16x2, 20x4). PDFs and schematics included in this folder.

## Key points
- Parallel interface (4-bit or 8-bit mode)
- Pins: VSS (GND), VDD (VCC), VO (contrast), RS, RW, E, D0â€“D7, A/K (backlight)
- Commonly used in 4-bit mode to save pins

See `HD44780.pdf` and included example application notes for wiring and initialization sequences.
