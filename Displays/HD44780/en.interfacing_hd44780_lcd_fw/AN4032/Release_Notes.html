Main Changes

  • First AN4032 "Interfacing an HD44780 2-Line LCD display with the STM8SVLDISCOVERY" firmware package delivery
  • Includes HD44780_template suited to LCD based application developments
  • Firmware available for STVD (Cosmic and Raisonance) and IAR EWSTM8
