## CJMCU-Beetle USB ATMEGA32U4 Mini Development Board For Arduino Leonardo

## Parameters

* Microcontroller: ATmega32U4
* Clock Speed: 16 MHz
* Operating Voltage: 5V DC
* Digital I/O Pins: 10
* PWM Channels: 4
* Analog Input Channels: 5
* UART: 1
* I2C: 1
* Micro USB: 1
* Flash Memory: 32 KB of which 4KB used by bootloader
* SRAM: 2.5 KB
* EEPROM: 1 KB
