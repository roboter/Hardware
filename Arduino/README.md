# Arduino

This directory contains various Arduino-compatible development boards and related projects.

Each board folder may include a `wiring.md` file with wiring notes and safety tips.

## Boards

### CJMCU-Beetle USB ATMEGA32U4
Mini development board based on ATMEGA32U4 (same chip as Arduino Leonardo). Includes PlatformIO projects for I2C scanner and NRF24L01 I2C interface.

### Digistump ATTINY85
ATTINY85-based development board with Arduino compatibility. Contains Neopixel examples and PlatformIO projects.
<br/><img src="Digistump ATTINY85/attiny.jpg" alt="Digistump ATTINY85" width="128">

### frankenstein
Custom Arduino-based projects and experiments.

### hw-260 development board
Development board with ISP programming capability. Includes basic blink example.

### leonardo
Arduino Leonardo board resources.

### nano v3
Arduino Nano V3 compatible board with CH340 USB interface. Includes official schematics and pinout documentation.
<br/><img src="nano v3/Arduino-Nano-CH340-USB-Mini-6-e1715369712116-1024x580.jpg" alt="Arduino Nano V3" width="256">

### PRO Mini
Arduino Pro Mini board (3.3V variant). Contains serial communication tools and examples.

### TestArduinoISP
Arduino ISP programmer configuration and test code for programming ATmega microcontrollers.

### YL-34
ATmega16/ATmega32 minimum system development board with onboard programmer.
