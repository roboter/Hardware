# WeAct Black Pill / BlackPill

A compact STM32F1-family development board ("Black Pill"). Common variants include STM32F103 and STM32F401-based versions.

Features:
- Small form-factor Cortex-M MCUs
- SWD header for programming/debugging
- Popular in breadboard projects and minimal system prototypes
