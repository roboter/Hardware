# WeAct Black Pill (STM32F401/F411)

See [wiring.md](wiring.md) for wiring and programming details.

## Features
- **CPU**: STM32F401CEU6 or STM32F411CEU6 (Cortex-M4)
- **Clock**: 84MHz (F401) / 100MHz (F411)
- **Flash**: 256KB (F401) / 512KB (F411)
- **RAM**: 64KB (F401) / 128KB (F411)
- **User LED**: PC13
- **USB**: USB-C for power and DFU programming
