# STM32

This directory contains STM32 microcontroller development boards, projects, and resources.

See [wiring.md](wiring.md) for general STM32 wiring and programming notes.

## Contents

### 32F3DISCOVERY_F303
STM32F3DISCOVERY board based on STM32F303CB (ARM Cortex-M4). Contains IAR projects and Microsoft development tools.

### 32F746GDISCOVERY
STM32F746G Discovery board with LTDC display controller. Includes:
- STM32CubeIDE projects
- Embeetle projects with LVGL port
- Display examples and documentation

### BlackPill
STM32F401-based "Black Pill" board with libOpenCM3 examples.

### BluePill
STM32F103C8T6 "Blue Pill" board. Contains:
- Arduino examples
- STM32CubeIDE projects
- PlatformIO examples
- Display drivers (Nokia 5110, ST7735)
- Neopixel examples

### DRG_STM32H750VBT6
DRG STM32H750VBT6 development board. Contains:
- STM32CubeIDE projects
- Keil MDK projects
- LED and display examples
- ADC examples

### MINI DEBUGGER
WeAct Mini Debugger (DAPLink/STLink V2.1) - USB-to-SWD debugger module.

### Mini STM32F103C8T6
Mini STM32F103C8T6 development board with PlatformIO examples.
<br/><img src="Mini STM32F103C8T6/S01aa2a9294b94205a584160c50bd861fL.webp" alt="Mini STM32F103C8T6" width="256">

### maple mini clone
Maple Mini clone based on STM32F103C8T6. Contains:
- PlatformIO projects
- Nokia PCD8544 display examples
- Bootloader files

### NUCLEO-C031C6
STM32 Nucleo-64 board with STM32C031C6 (ARM Cortex-M0+). Includes CubeIDE SSD1306 OLED examples.

### NUCLEO-F303K8
STM32 Nucleo-32 board with STM32F303K8 (ARM Cortex-M4). Contains:
- CubeIDE projects (PWM, Neopixel)
- Application notes

### NUCLEO-G431KB
STM32 Nucleo-32 board with STM32G431KB documentation.

### NUCLEO-H563ZI
STM32 Nucleo-144 board with STM32H563ZI (ARM Cortex-M33). Includes user manuals and schematics.

### NUCLEO-L152RE
STM32 Nucleo-64 board with STM32L152RE (ARM Cortex-M3, low-power). Contains CubeIDE projects.

### STLink
STLink v2 programmer/debugger images and documentation.
<br/><img src="STLink/st-linkv2-st-link-v2cn-st-link-stlink-programmer-stm8-stm32.jpg" alt="STLink v2" width="256">

### STM32 Mini V2.0
STM32 Mini V2.0 development board with display and NRF24L01 examples.

### STM32F103C8T6_DEV
Extensive collection of STM32F103C8T6 development board examples including:
- Multiple versions and variants
- PlatformIO projects
- Display drivers
- USB communication examples

### STM32F103C8T6-STM32-Smart-V2_0
STM32 Smart V2.0 development board with display examples and AT24xx EEPROM projects.

### STM32F407VET6 black board
STM32F407VET6 development board with CubeIDE projects for CC1101 and NRF24L01 communication.

### STM32F429-Discovery
STM32F429 Discovery board with:
- STM32CubeIDE examples
- Embeetle projects with LVGL demos
- Display controller examples

### STM32F3DISCOVERY
STM32F3DISCOVERY board documentation.

### WeAct Black Pill V3.0
WeAct Black Pill V3.0 (STM32F401) with PlatformIO and CubeIDE examples.

### Documentation
Various STM32 documentation files including:
- STLink derivatives overview
- HAL and low-layer drivers documentation
- Discovery board getting started guides
