# STM32WB55 Nucleo Pack (P-NUCLEO-WB55)

This directory contains resources for the P-NUCLEO-WB55 pack, which includes a Nucleo-64 board and a USB dongle based on the STM32WB55 microcontroller.

See [wiring.md](wiring.md) for wiring and programming details.

## Features
- **Dual-core Architecture**: ARM Cortex-M4 (application) and ARM Cortex-M0+ (wireless stack).
- **Wireless Protocols**: Bluetooth 5.3, Zigbee 3.0, OpenThread.
- **Nucleo-64 Board**: Features an integrated PCB antenna and ST-LINK/V2-1.
- **USB Dongle**: Compact form factor with integrated PCB antenna.
