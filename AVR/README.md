# AVR

This directory contains AVR microcontroller development boards and programming tools.

## Contents

### Butterfly
AVR Butterfly development board - an educational evaluation board with LCD, sensors, and USB interface. Contains:
- AVR109 bootloader documentation
- Butterfly application firmware
- Programming guides and documentation

### STK500
STK500 AVR development kit documentation and firmware. The STK500 is a full-featured development board and programmer for AVR microcontrollers.
- STK500 communication protocol documentation
- Application notes and example code
