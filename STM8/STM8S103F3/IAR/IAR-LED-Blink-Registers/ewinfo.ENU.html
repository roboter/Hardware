STM8L-Discovery Simple LED Blink Example

This example uses the STM8L-Discovery board with the STM8L152C6 device and demonstrates how to make the LED blink.

Follow the steps below to run the example on your hardware.

  • Connect the computer and the STM8L-Discovery board using a USB cable.
  • Open the project.
  • Choose Project>Download and Debug. This will build and download your application to the STM8L-Discovery board, and then start the debugging session.
  • To execute the application in the debugger, choose Debug>Go.
