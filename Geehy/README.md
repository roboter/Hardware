# Geehy

This directory contains Geehy APM32 microcontroller development boards and SDKs.

## Contents

### APM32F072VB
APM32F072VB evaluation board and development resources. APM32F072VB is a Cortex-M0 based microcontroller from Geehy (pin-compatible with STM32F072).

<br/><img src="APM32F072VB/Geehy-APM32F072VB-Mini-Board-1-361x550.webp" alt="APM32F072VB Mini Board" width="256">

Contains:
- Evaluation board user manuals
- Mini board documentation
- SDK v1.7
- SEGGER Embedded Studio project examples
- Datasheets and reference manuals
