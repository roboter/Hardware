# Geehy APM32 Wiring and Programming

Geehy APM32 microcontrollers are largely pin-compatible and software-compatible with STM32 families.

## Programming
- **Interface**: Uses the same SWD (Serial Wire Debug) interface as STM32.
- **Tools**: Compatible with ST-Link, J-Link, and DAPLink.
- **Protocol**: ARM Cortex-M0/M3/M4 standard.

## Specific Boards
- [APM32F072VB/wiring.md](APM32F072VB/wiring.md): Details for the APM32F072VB Mini Board.

## Connection Reference (SWD)
| Pin | Function |
| --- | --- |
| SWDIO | Serial Wire Data |
| SWCLK | Serial Wire Clock |
| GND | Ground |
| 3V3 | Power |
