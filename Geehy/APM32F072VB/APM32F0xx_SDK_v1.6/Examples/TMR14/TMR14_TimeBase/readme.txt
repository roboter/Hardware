&par Example Description

This example aims to show how to realize timing one second by using TMR14 peripheral  generating time base .
LED2 and LED3 will toggle pre second.

&par Directory contents

  - TMR/TMR14/TMR14_TimeBase/apm32f0xx_int.c           Interrupt handlers
  - TMR/TMR14/TMR14_TimeBase/Source/main.c              Main program

&par Hardware and Software environment

  - This example runs on APM32F030 or APM32F072 MINI Devices.
