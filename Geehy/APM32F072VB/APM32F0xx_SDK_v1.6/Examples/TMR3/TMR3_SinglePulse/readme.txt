&par Example Description

This example shows how to use the TIM peripheral to generate a One pulse Mode
after a falling edge of an external signal is received in Timer Input pin.
The TIM3 waveform can be displayed using an oscilloscope.

using KEY1 to generate onepulse

&par Directory contents

  - TMR/TMR3/TMR3_SinglePulse/apm32f0xx_int.c            Interrupt handlers
  - TMR/TMR3/TMR3_SinglePulse/Source/main.c              Main program

&par Hardware and Software environment

  - This example runs on APM32F030 or APM32F072 MINI Devices.
