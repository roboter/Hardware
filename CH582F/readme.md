# CH582F

This directory contains resources for the CH582F microcontroller development board.

## Contents

CH582F is a BLE (Bluetooth Low Energy) microcontroller from WCH (Nanjing Qinheng Microelectronics).

For more information, see the [WeActStudio WCH BLE Core repository](https://github.com/WeActStudio/WeActStudio.WCH-BLE-Core).
